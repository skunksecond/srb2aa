# srb2aa
extremely simple python script that adds SRB2 addons to autoexec.cfg so the game auto-loads addons at start up.
(Sonic Robo Blast 2 Addon Automator)

# Usage
ensure you have python installed on your computer, then drag the srb2aa.py file to the root directory of Sonic Robo Blast 2's folder, and double click to generate your autoexec.cfg file.
